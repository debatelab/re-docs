ReflectiveEquilibrium¶

class basics.ReflectiveEquilibrium¶

Bases: abc.ABC

Class representing a process of finding RE states.

Implementations of this abstract class represent a formal model of a process that finds reflective equilibria (RE states). The process of finding an RE state starts with an initial epistemic state (initial commitments \(\mathcal{C}_0\)), which are step-wise adjusted until they reach an RE state. An RE state is an epistemic state - pairs of commitments and theories \((\mathcal{C}, \mathcal{T}\)) - that maximises an achievement function \(Z\) and is dialectically consistent.

ReflectiveEquilibrium.Results¶

class basics.ReflectiveEquilibrium.Results(*args, **kwargs)¶

Bases: dict

Result class for RE processes as a typed dictionary. It contains the following entries:

Variables

  • initial_commitments – A position representing the initial commitments.

  • weights – Dictionary for the weights. It has entries for ‘account’, ‘systematicity’ and ‘faithfulness.

  • theory – A position representing the theory in the fixed point reached by the RE process.

  • commitments – A position representing the commitments in the fixed point reached by the RE process.

  • evolution (theory) – A list of positions storing the chosen theories for each theory adjustment step.

  • commitments_evolution – A list of positions storing the chosen commitments for each commitment adjustment step.

  • achievements – A list of floats for the value of the achievement function after each step.

  • choices (random) – A list of numbers representing steps in which a random choice among equally well performing candidates occured.

  • branches – A list of further Result dicts (including all information from above except for ‘branches’, which are set to None) for every alternative path due to random choices.
