ReflectiveEquilibrium¶

class basics.ReflectiveEquilibrium(dialectical_structure=None, initial_commitments=None, model_name=None)¶

Bases: abc.ABC

Class representing a process of finding RE states.

Implementations of this abstract class represent a formal model of a process that finds reflective equilibria (RE states). The process of finding an RE state starts with an initial epistemic state (initial commitments \(\mathcal{C}_0\)), which are step-wise adjusted until they reach an RE state. An RE state is an epistemic state - pairs of commitments and theories \((\mathcal{C}, \mathcal{T}\)) - that maximises an achievement function \(Z\) and is dialectically consistent.
