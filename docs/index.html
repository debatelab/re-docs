A Formal Model of Reflective Equilibrium¶

A Python implementation of a reflective equilibrium model based on [BBB], [Betz2010] and [Betz2012].

Installation¶

The RE Model consists of two python packages – remodeldescription and remodel. If you want to use and/or extend the model implementation you have to install both packages locally, by

  • first git-cloning the repositories of both packages:

    • git clone git@github.com:debatelab/re-python.git

    • git clone git@github.com:debatelab/re-model-description.git

  • and then installing the packages by running pip install -e . from the local directories of the packages (e.g. local-path-to-repository/remodel). Pip will install the packges by using the setup.py file. (The -e-option will install the packages in the editable mode, allowing you to change the source code.)

Note

Both packages require a python version >= 3.8. The package remodel further depends on the packages bitarray, numba and PySat (which will be installed when installing remodel by using pip).

Tutorials¶

The are different Jupyter notebooks that provide step-by-step instructions of running and adapting the model:

  • intro-repython.ipynb: Introduction of using re-python. Covering the following topics:

    • Running the model

    • Export to JSON

    • Extending the model

  • ensemble-generation-tutorial.ipynb: A notebook that illustrates how to generate model ensembles with the ensemble generation module.
